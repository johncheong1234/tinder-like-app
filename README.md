# Tinder-Like App

This project sets up a Tinder-like application with a mock API server. Follow the steps below to get the application running on your local machine.

## Prerequisites

- Node.js
- npm

## Setup Instructions

1. Clone the repository:

    ```sh
    git clone https://github.com/johncheong1234/tinder-like-app.git
    cd tinder-like-app
    ```

2. Install the required dependencies:

    ```sh
    npm install
    ```

3. Start the application:

    ```sh
    npm run start
    ```

4. Open another terminal and start the mock API server:

    ```sh
    npm run mock-api
    ```

The application should now be running, and you can access it in your web browser.
